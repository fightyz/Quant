# jisilu-digest

Claude Code 插件：自动抓取并整理[集思录](https://www.jisilu.cn/explore/)最新投资讨论，生成结构化摘要。

## 功能

- 抓取集思录论坛最新帖子列表，按时间窗口筛选
- 按投资价值分 A/B 档，优先深入分析高价值帖子
- 自动下载并分析帖子中的图片（数据图表、K线图等）
- 输出按"宏观信号 / 品种策略 / 资金渠道 / 投资警示"分类的结构化摘要
- 每个条目附原帖链接、关键数据、多方观点

## 安装

```bash
/plugin install https://github.com/fightyz/jisilu-digest
```

## 使用

安装后直接对话即可触发：

- "帮我看看集思录最近有什么值得关注的讨论"
- "集思录最近关于转债的讨论有哪些？"
- "https://www.jisilu.cn/explore/ 回顾最近 3 天有更新的帖子"

也可以通过斜杠命令调用：`/jisilu-digest:jisilu-digest`

## 默认时间窗口

- 帖子筛选：运行当天及前一天（两个完整自然日）
- 回复回顾：每个帖子最近 2 周内的回复

用户可在提示中自定义时间范围。
